# Cabinet de Réflexologie de l'Audomarois

Site vitrine pour **Caroline Saintenoy**, réflexologue à Nort-Leulinghem (62), spécialisée en cancérologie et handicap.

🌐 **Site en production** : [reflexologie-audomarois.com](https://reflexologie-audomarois.com/)

## Cliente

Caroline Saintenoy — Réflexologue certifiée
- Cabinet à Nort-Leulinghem (Audomarois, près de Saint-Omer)
- Spécialités : réflexologie plantaire, onco-réflexologie, auriculo-réflexologie, réflexologie adaptée au handicap

## Stack technique

Site statique 100 % HTML / CSS / JavaScript vanilla, sans framework ni build system.

- **Typographies** : Cormorant Garamond + Nunito Sans (Google Fonts)
- **Palette organique** sur-mesure : sauge, lin, argile, mousse
- **SEO** : Schema.org `HealthAndBeautyBusiness`, Open Graph, sitemap, balises canoniques
- **Performance** : preconnect, optimisation des images, single-page
